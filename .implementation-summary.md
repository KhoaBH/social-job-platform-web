# API PUT Skill User - Tóm Tắt Triển Khai

## 📋 Mô Tả
Đã thiết kế và triển khai đầy đủ chức năng chỉnh sửa kỹ năng người dùng với API endpoint `PUT /skills/user/{id}`.

## 🔧 Các Thay Đổi Chi Tiết

### 1. Backend API Setup (`profileApi.ts`)
- ✅ **Thêm `UpdateUserSkillPayload` type** trong `types.ts`
  ```typescript
  interface UpdateUserSkillPayload {
    skillId: string;
    level: number;
  }
  ```

- ✅ **Tạo 2 API endpoint mới:**
  - `updateUserSkill`: PUT `/skills/user/{skillId}`
    - Payload: `{ userId: string; skillId: string; body: UpdateUserSkillPayload }`
    - Method: PUT
    - Response: `BackendUserSkill`
    - Invalidates tags: `{ type: "ProfileSkill", id: USER_${userId} }`
  
  - `deleteUserSkill`: DELETE `/skills/user/{skillId}`
    - Params: `{ userId: string; skillId: string }`
    - Method: DELETE
    - Invalidates tags: `{ type: "ProfileSkill", id: USER_${userId} }`

- ✅ **Export hooks:**
  - `useUpdateUserSkillMutation`
  - `useDeleteUserSkillMutation`

### 2. UI Components

#### AddSkillModal (`components/models/AddSkillModal/`)
- ✅ **Cập nhật types.ts** để thêm props cho edit mode:
  ```typescript
  initialData?: ProfileSkillView;
  onDelete?: (skillId: string) => Promise<void>;
  isDeleting?: boolean;
  ```

- ✅ **Cập nhật index.tsx** để hỗ trợ:
  - Hiển thị modal với title "Sửa kỹ năng" hoặc "Thêm kỹ năng"
  - Pre-fill form khi edit: `skillId` và `level`
  - Xóa skill với confirmation dialog
  - Button "Cập nhật" hoặc "Lưu" tùy theo mode
  - Xử lý error khi xóa

#### SkillItem Component (`components/shared/SkillItem.tsx`)
- ✅ **Tạo component mới** (giống ExperienceItem):
  - Hiển thị: Tên kỹ năng, danh mục, level (1-5)
  - Click để edit: `onClick={() => onEdit?.(skill)}`
  - Hover effect: `hover:bg-gray-50`
  - Props: `skill: ProfileSkillView`, `onEdit?: callback`

#### ProfileSkillsTab (`components/tabs/ProfileSkillsTab.tsx`)
- ✅ **Thêm state:**
  - `editingSkill: ProfileSkillView | undefined`

- ✅ **Thêm handlers:**
  - `handleEditSkill(skill)`: Đặt skill vào edit mode, mở modal
  - `handleUpdateSkill(data)`: Gọi API PUT, refresh list
  - `handleDeleteSkill(skillId)`: Gọi API DELETE, refresh list
  - `handleAddSkill(data)`: Existing handler cho create

- ✅ **Cập nhật UI:**
  - Thay thế skill list từ manual div sang `<SkillItem />`
  - Pass `onEdit` callback cho SkillItem
  - Cập nhật AddSkillModal với tất cả props mới

- ✅ **Thêm props type:**
  ```typescript
  onUpdateUserSkill?: (skillId: string, data: AddSkillFormData) => Promise<void>;
  isUpdatingUserSkill?: boolean;
  onDeleteUserSkill?: (skillId: string) => Promise<void>;
  isDeletingUserSkill?: boolean;
  ```

#### Profilelayout (`components/Profilelayout.tsx`)
- ✅ **Import mutations:**
  - `useUpdateUserSkillMutation`
  - `useDeleteUserSkillMutation`

- ✅ **Khai báo hooks:**
  ```typescript
  const [updateUserSkill, { isLoading: isUpdatingUserSkill }] = useUpdateUserSkillMutation();
  const [deleteUserSkill, { isLoading: isDeletingUserSkill }] = useDeleteUserSkillMutation();
  ```

- ✅ **Tạo handlers:**
  - `handleUpdateUserSkill(skillId, payload)`: Gọi API PUT
  - `handleDeleteUserSkill(skillId)`: Gọi API DELETE

- ✅ **Pass props đến ProfileSkillsTab:**
  - `onUpdateUserSkill={handleUpdateUserSkill}`
  - `isUpdatingUserSkill={isUpdatingUserSkill}`
  - `onDeleteUserSkill={handleDeleteUserSkill}`
  - `isDeletingUserSkill={isDeletingUserSkill}`

## 📊 API Endpoint Chi Tiết

### PUT `/skills/user/{id}`
**Request Body:**
```json
{
  "skillId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "level": 0
}
```

**Response:**
```json
{
  "id": "...",
  "level": 0,
  "skill": {
    "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "name": "React",
    "category": "Frontend"
  }
}
```

### DELETE `/skills/user/{id}`
**Status:** 204 No Content

## ✨ Flow Sử Dụng

### Thêm Kỹ Năng
1. Nhấn "+ Thêm kỹ năng"
2. Modal AddSkillModal mở với title "Thêm kỹ năng"
3. Chọn skill + level
4. Nhấn "Lưu"
5. API POST `/skills/user` được gọi
6. List refresh

### Chỉnh Sửa Kỹ Năng
1. Nhấn vào item skill
2. Modal AddSkillModal mở với:
   - Title "Sửa kỹ năng"
   - Form pre-filled: skillId, level
   - Nút "Xóa" (left)
   - Nút "Cập nhật" (right)
3. Chỉnh sửa + nhấn "Cập nhật"
4. API PUT `/skills/user/{skillId}` được gọi
5. List refresh

### Xóa Kỹ Năng
1. Chỉnh sửa skill (bước trên)
2. Nhấn "Xóa" button
3. Confirm dialog hiện lên
4. Nhấn "Xóa" để confirm
5. API DELETE `/skills/user/{skillId}` được gọi
6. Modal đóng, list refresh
